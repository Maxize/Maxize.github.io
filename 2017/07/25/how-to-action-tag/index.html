添加标签

Git 使用的标签有两种类型：轻量级的（lightweight）和含附注的（annotated）。

一般我们都建议使用含附注型的标签，有自身的校验和信息，包含着标签的名字，电子邮件地址和日期，以及标签说明，存储的一个独立对象，两者的操作如下：
